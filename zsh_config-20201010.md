# zsh_config-20201010

## 1. 安装zsh

实际上manjaro默认已经安装。运行` cat /etc/shells` 查看有没有zsh，结果如下：

```shell
/bin/sh
/bin/bash
/bin/zsh
/usr/bin/zsh
/usr/bin/git-shell
```

设置zsh为默认shell

```sh
chsh -s $(which zsh)
```

重启电脑

查看默认的shell是不是真的设置成zsh，执行`echo $SHELL`，运行结果如下

``` shell
echo $SHELL
/usr/bin/zsh
```

## 2. 安装oh my zsh

```shell
$ sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```

```shell
❯ echo $ZSH_CUSTOM
/home/loulvlin/.oh-my-zsh/custom
```

**zsh-autosuggestions 插件安装**

```bash
git clone git://github.com/zsh-users/zsh-autosuggestions $ZSH_CUSTOM/plugins/zsh-autosuggestions
```

**zsh-syntax-highlighting 插件安装**

```bash
git clone https://github.com/zsh-users/zsh-syntax-highlighting.git ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting
```

**zsh-completions 插件安装**

```bash
git clone https://github.com/zsh-users/zsh-completions ~/.oh-my-zsh/custom/plugins/zsh-completions
```

在 `~/.zshrc` 中配置

```
plugins=(
   git
   zsh-syntax-highlighting
   zsh-autosuggestions
   zsh-completions
)
```

**autojump 插件安装**

```bash
cd /home/soft
git clone git://github.com/joelthelion/autojump.git
```

`cd autojump`，执行

```bash
./install.py
```

在`~/.zshrc`末尾加入

> 需要参考安装后的提示信息

```shell
[[ -s /home/loulvlin/.autojump/etc/profile.d/autojump.sh ]] && source /home/loulvlin/.autojump/etc/profile.d/autojump.sh 
autoload -U compinit && compinit -u
```

```shell
source .zshrc
```
